# \[unwrap]

Running:

```bash
# Bun
bunx @templ8/unwrap --help

# Node
npx @templ8/unwrap --help
```

## Commands

- `init` - Initialize a new project.
- `help` - Display help information.
- `license` - Generate a license file.

## Development

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

### Linking

```bash
bun link
```

Now you can run `unwrap` from anywhere.
